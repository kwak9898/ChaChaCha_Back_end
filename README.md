## 🤷‍♀️ 왜 26인가요?

저희들은 항해99에서 26조로 만나게 되었습니다. 캠프 초반부터 26(이륙)하여, 앞으로의 폭풍성장을 기원하는 마음으로 짓게되었습니다.
